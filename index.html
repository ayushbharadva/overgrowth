Every commit feeds it. Type a GitHub username and grow the tree that years of late nights, languages and abandoned side projects deserve.

How to read a tree

  • Height & depth — years on GitHub, repos built
  • Trunk — how actively they push
  • Branching & colors — language diversity
  • Glowing blossoms — stars earned
  • Bare grey branches — abandoned repos, the honest scars
  • The lean — late-night pushes bend it toward the dark
  • Leaf color — green when fresh, brown when dormant
  • Same name, same tree — it's deterministic; it's yours
